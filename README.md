# Python Dark Calc

Простой и стильный калькулятор на **Python** с использованием библиотеки **Tkinter**.

##  Особенности 
*   Dark Mode: Стильный интерфейс в тёмных тонах.
*   Математика: Базовые операции, возведение в степень (`**`), извлечение квадратного корня (`√`) и число Пи (`π`).
*   Удобство: Кнопка Backspace (`⌫`) для удаления последней цифры и большая кнопка `C` для полной очистки.
*   Обработка ошибок: Защита от «дичи» при вводе некорректных примеров.

## Как запустить
1. Установите Python.
2. Скачайте файл `main.py`.
3. Запустите его командой:
   ```bash
   python main.py
   ```

##  Технологии
*   Python 3.x
*   Tkinter
