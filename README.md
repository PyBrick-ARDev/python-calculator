# Python Dark Calc

Простой и стильный калькулятор на **Python** с использованием библиотеки **Tkinter**.

##  Особенности
*   **Dark Mode:** Интерфейс в тёмных тонах.
*   **Функции:** Сложение, вычитание, умножение, деление и возведение в степень (`**`).
*   **Очистка:** Отдельная кнопка "C" для быстрого сброса результата.
*   **Обработка ошибок:** Выводит предупреждение при некорректном вводе.

## Как запустить
1. Установите Python.
2. Скачайте файл `main.py`.
3. Запустите его командой:
   ```bash
   python main.py
   ```

##  Технологии
*   Python 3.x
*   Tkinter
